FAQ

Practical limits

Does this upload my Markdown anywhere?

No. The conversion happens in the browser on the current device.

Will complex Markdown always match Pandoc output?

No. This is a browser renderer, so results are designed to look good, not to exactly match a CLI pipeline.

Can I host this without a server?

Yes. It is a static site and can be hosted directly on GitHub Pages or any static file host.

What is the main limitation?

Very large documents or image-heavy content can be slower because PDF rendering happens fully in the browser.
